Python fungsi

Pada saat kita membuat sebuah program besar dan memiliki banyak fitur, kita diharuskan membuat sebuah fungsi kalau tidak menggunakan fungsi? tentu kita akan kerepotan menulis kode programnya, karena banyak yang harus ditulis dan kode akan sulit untuk dibaca dengan menggunakan fungsi kita bisa memecah program besar menjadi program yang lebih sederhana.

membuat fungsi di python

membuat fungsi di python dibuat dengan kata kunci defContoh:


  def nama_fungsi():
      # isi fungsi/code...
      print "ini dari fungsi"

biar lebih jelas coba buat file bernama fungsi tes_fungsi.py dan ikuti kode berikut


  def nama_saya():
      print "nama saya adalah muslim ambari :)"

save, dan jalankan

kenapa tidak muncul apa-apa? karena belum kita panggil fungsinya :D lalu bagaimana cara memanggil nya? tinggal tulis saja nama fungsinya dan diakhiri dengan tanda kurung ()sehingga menjadi seperti ini:


  # membuat fungsi
  def nama_saya():
      print "nama saya adalah muslim ambari :)"

  # memanggil fungsi
  nama_saya()

Nah, sekarang baru muncul :D

Fungsi dengan parameter

parameter sama seperti variabel, tungasnya untuk menyimpan data. parameter dibuat didalam tanda kurung (nama_variabel)Contoh: 


  def saya(nama):
      print nama
  
  saya("muslim ambari")

bisa dilihat, bahwa kita membuat sebuah parameter yang berisikan variabel nama. lalu kita panggil fungsinya dan mengisikan nilai kedalam variabel nama tersebut dengan "muslim ambari",jadi "muslim ambari" itu adalah nilai yang kita berikan, ika terdapat lebih dari dua parameter bisa dipisah dengan tanda koma ( , ),Contoh: 


  def data_saya(nama, umur):
      print nama
      print umur
  
  data_saya("muslim ambari",18)

kita membuat 2 parameter yang berisikan variabel nama dan umuruntuk variabel nama kita isi dengan "muslim ambari" sedangkan untuk variabel umur kita isi dengan 18Jika dijalankan, Hasilnya:

memberikan nilai default pada parameter, nilai default ini berfungsi jika pengguna tidak menyediakan nilai argumen saat fungsi dipanggil Contoh: 


  def contoh(nilai = 10):
      print nilai
  
  # tidak memberikan nilai parameter
  contoh()

  # memberikan nilai parameter
  contoh(19)

Hasilnya:

perhatikan, saat kita tidak memberikan nilai pada parameter, akan tampil nilai default yang kita berikan yaitu 10 kemudian kita memenggil fungsi lagi dan membarikan nilai 19, maka otomatis nilai default 10 akan diganti dengan 19 oke, cukup sekian tentang pembahasan fungsi, semoga kamu sudah paham dan silahkan pelajari tutorial selanjutnya Lanjut
