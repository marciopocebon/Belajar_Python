membuat promgram python pertama

pada tutorial ini kita akan membuat program python pertama kamu pastikan sudah menginstall python terlebih dahulu, kalau belum install silahkan install dulu. dan untuk pengguna linux, tidak perlu install lagi, karena python sudah terinstall secara default di system operasi Linux, tapi kalau mau install ya... install aja gak ada yg larang kok :D. oke, mari kita buat program python pertama kamu buka text editor, saya sarankan menggunkan 'sublime text' atau 'Visual Studio Code' biar lebih mantap ngodingnya :D jika sudah siap, silahkan buat file baru bernama helloWorld.py dan ikuti kode berikut 

            
  print "hello world"

buka terminal dan masuk ke directory dimana file helloWorld.py tadi dibuat, lalu jalankan dengan perintah 'python2 helloWorld.py' (tanpa tanda kutip). dan lihat hasilnya
