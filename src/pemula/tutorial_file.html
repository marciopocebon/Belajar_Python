Membaca dan Menulis File di Python

proses baca dan tulis sangat penting untuk dipahami, dalam pemrograman Python, karena banyak digunakan untuk pengolahan dan pemerosesan file.Pada pembelajaran kali ini, kita akan belajar cara membaca dan menulis kesuatu file.

Membaca file di python

python sudah membawa fungsi open() untuk memaca dan menulis file fungsi ini memiliki dua parameter yaitu nama file dan mode nya, penulisan fungsi open() ini sama seperti variabel kita harus memberikan nama variabelnya untuk menampung isi file

Contoh penulisan:


  file = open("path/ke/file.txt","r")

parameter mode fungsinya untuk menentukan hak akses terhadap file mode yang tersedia di python:

keterangan mode
membuka file hanya untuk dibaca r
untuk menulis ke file, jika isi file sudah ada.. ada maka akan diganti dengan yang baru ditulis w
untuk menambahkan data/tulisan ke file, jika isi file sudah ada maka akan ditambahkan dan tidak diganti a
untuk membaca dan menulis ke file r+

membaca file perbaris

untuk membaca file perbaris, kita membutuhkan method readlines()Coba dalam program: 


  f = open("data.txt","r")
  print f.readlines()

  f.close()

  f = open("data.txt","r")

Menulis ke file

python sudah menyediakan fungsi write() untuk menulis text ke file, dan seperti yang sudah kita ketahui, ada tiga mode yang digunakan bila ingin menulis file, yaitu: w, a, dan r+ bagaimana cara menggunakan 3 mode tersebut? mari kita plajari menulis ke file dengan mode "W"silahkan buat file baru bernama latihan.py dan ikuti kode berikut 


  # buka file
  f = open("file.txt","w")
  
  text = "hello file\n dari python"
  
  # menulis ke file
  f.write(text)

  # menutup file
  f.close()

jika dijalankan tidak muncul apa-apa...

kenapa? karena kita hanya menulis ke file saja dan tidak menampilkan pesan apapun,tapi.. coba buka file.txt maka akan ada tulisan yang kita tulis tadi

menulis ke file dengan mode "a"seperti yang sudah kita pahami bahwa mode "a" ini untuk menambahkan tulisan ke file,artinya jika isi file sudah ada maka akan ditambahkan dan tidak digantisekarang mari kita latihan... buat 2 buah file baru dalam folder yang sama seperti ini


  tulis_file.py
  sata_saya.txt

silahkan isi file yang bernama data_saya.txt seperti ini:


  satu
  dua

dan untuk file.py silakan isi dengan kode berikut ini:


  f = open("file.txt","a")

  text = ["\ntiga","\nempat","\nlima"]

  f.writelines(text)
  f.close()

perhatikan bahwa fungsi writelines() untuk menuliskan sebuah list ke file.sekarang silahkan jalankan

seharusnya tidak muncul apa-apa karena kita hanya menulis ke file saja, tapi coba buka file yang bernama sata_saya.txt

maka tulisanya akan ditambahkan dan tidak diganti karena kita menggunakan mode "a" oke, cukup sekian tentang pembelajaran kali ini, semoga kamu sudah mengerti
