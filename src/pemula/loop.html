memahami loop pada python

sekarang kita akan belajar loop di python, apa itu loop?loop itu adalah perulangan, biar lebih sreg kita sebut saja perulangan perulangan dalam bahasa pemograman berfungsi untuk menyuruh program melakukan sesuatu secara berulang ulang, ada dua macam perulangan di python, diantaranya:

1.perulangan for2.perulangan while

1. perulangan for

penulisan perulangan for:


  for index in range(jumlah_perulangan):
    # kode...
    # ...

Contoh: 


  jumlah = 5

  for i in range(jumlah):
    print "perulangan ke:",i

note: variabel jumlah untuk menentukan jumlah perulangan,Hasilnya:

menampilkan data list menggunakan perulangan for 


  buah = ["melon","dukuh","rambutan","durian"] # enak ya :D

  for hasil in buah:
    print hasil

2. perulangan while

bentuk penulisan perulangan while 


  while( #kondisi ):
    # code ...
    # ...

Contoh:


  hitung = 0

  while(True):
    hitung += 1
    jawab = raw_input("lagi?: ")
    if jawab != "ya":
      break
  
  print "perulangan sebanyak {} kali".format(hitung)

Pertama menentukan variabel untuk menghitung,dan menentukan kapan perulangan berhenti, jika user menjawab selain ya... maka proses looping akan diberhentikan oleh fungsi break,setelah perulangan selesai, cetak berapa kali perulangan terjadi 


    print "perulangan sebanyak {} kali".format(hitung)
