aturan penulisan syntak di python

kali ini kita akan belajar cara penulisan syntak yang wajib diketahui di python. apa itu syntak ? syntak adalah aturan penulisan blok kode pada program. penulisan syntak pada bahasa pemograman python harus dipatuhi. karena python sangat cerewet dalam urusan syntak jika tidak dipatuhi ?maka program akan error di python ada 5 aturan penulisan syntak. apa saja itu?mari kita bahas semuanya.

1.Penulisan Statement

Statement adalah sebuah intruksi atau kalimat perintah yang akan dieksekusi oleh komputer.Contoh 

              
  print "hello"
  print "kamu"

Penulisan satu statement tidak diakhiri dengan tanda titik-koma.Sedangkan, bila kita ingin menulis lebih dari satu statement dalam satu baris, maka kita harus memisahnya dengan titik-koma.Contoh 

            
  print "hello"; print "kamu"

tapi menurut beberapa style guide python, tidak dianjurkan menulis lebih dari satu statement dalam satu baris. Karena akan sulit dibaca.

2.Penulisan String

String adalah teks atau kumpulan dari karakter.String dalam pemrograman biasanya ditulis dengan dibungkus menggunakan tanda petik.Bisa menggunakan tanda petik tunggal maupun ganda.Contoh: 

            
  print "hello"
  print 'kamu'
  nama = "muslim ambari"
  umur = '18 tahun'

3.Penulisan Case

Sintak Python bersifat case sensitive, artinya namaSaya dengan namasaya dibedakan.Contoh: 

          
  namaSaya = "muslim ambari"
  NamaSaya = "muslim ambari"

Lihat antara variabel namaSaya dan NamaSaya itu beda. meskipun isi variabelnya sama.

4.Penulisan Blok Program

Blok program adalah kumpulan dari beberpaa statement yang digabungkan dalam satu blok.Penulisan blok program harus ditambahkan tab/spasi 2 atau 4 kali contoh penulisan yang benar

5.Penulisan Komentar

Komentar adalah baris kode yang tidak akan dieksekusi oleh prgram.Komentar digunakan untuk memberikan informasi tambahan dan untuk menonaktifkan kode.Ada beberapa cara menulis komentar pada pemrograman Python.

menggunakan tanda pagar (#)

komentar menggunakan tanda pagar (#)Cara ini paling sering digunakan.Contoh: 

					
  # ini adalah komentar
  # ini juga

menggunakan tanda petik

Selain untuk mengapit string, tanda petik juga dapat digunakan untuk membuat komentar.Contoh: 

						
  "komentar dengan tanda petik ganda"
  'komentar menggunakan tanda petik tunggal'

menggunakan triple tanda petik

triple tanda petik, sering digunakan untuk menuliskan dokumentasi Contoh: 

					
  """
  komentar dengan triple tanda petik
  """
