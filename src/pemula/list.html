python list

kali ini kita akan belajar list pada python, apa itu list ? List adalah struktur data pada python yang mampu menyimpan lebih dari satu data apa saja yang akan kita pelajari tentang list?

1. membuat list2. menampilkan data list3. menambahkan data kedalam list4. menghapus isi list5. menghapus data yang sama di list

1.membuat list

pembuatan type data list sama dengan membuat variabel biasabedanya, dalam membuat list kita tambahkan tanda kurung siku,Contoh: 


  buah = ["jeruk","apel","lemon"]

data yang akan kita simpan harus di capit dengan tanda kutip, dan dipisahkan dengan tanda koma.

2. menampilkan data list

sebelumnya harus diketahui bahwa index pada list selalu dimulai dari nol ( 0 ), sama seperti array biar lebih paham. mari kita praktekan, silahkan buat file baru bernama latihan_list.pydan ikuti code berikut 


  buah = ["jeruk","apel","lemon"]

  # menampilkan data jeruk, karena jeruk berada di dalam index ke 0
  print buah[0]

Hasilnya

3. menambahkan data kedalam list

untuk menambahkan data kedalam list kita bisa gunakan fungsi append(data) kita coba dalam program, buat file baru dan ikuti kode berikut 


  # membuat list kosong
  buah = []

  # tampilkan list sebelum ditambahkan
  print buah

  # menambahkan 1 data kedalam list
  buah.append("durian")

  # tampilkan list sesudah ditambahkan
  print buah

Hasilnya

lihat, sebelum ditambahkan tidak ada data di list(kosong), karena kita belum menambahkan data ke list. tapi sesuah kita tambahkan menggunakan fungsi append() maka akan tampil data durian yang kita tambahkan tadi

4. menghapus isi list

dan untuk menghapus isi list kita bisa gunakan fungsi del() Contoh penggunaan fungsi del 


  del(data[0])

data adalah nama variabelnya dan 0 adalah index data yang ingin kita hapus biar lebih paham, mari kita latihan. buat file baru dan ikuti kode berikut 


  # membuat list
  buah = ["apel","jeruk","durian"]
  
  # menghapus data jeruk, yang berada pada index ke 1
  del(buah[1])

  # menampilkan list buah
  print buah

Hasilnya

bisa dilihat bahwa data yang bernama jeruk tidak ada, karena sudah kita hapus menggunakan fungsi del() tadi

5. menghapus data yang sama di list

bagaimana jika kamu sedang membuat program untuk mengumpulkan banyak data sedangkan di tumpukan data tersebut banyak sekali data yang sama, maka kamu harus menghapusnya secara manual. apakah boleh seperti itu? ya boleh boleh saja, tapi bagaimana jika data yang kamu kumpulkan terdapat lebih dari 1000?,itu akan sangat melelahkan. maka dari itu saya akan mengajarkan kepada kamu, bagaimana cara mengapus data yang sama di dalam list menggunakan fungsi set() fungsi set ini bisa kita gunakan untuk menghapus data yg sama di dalam list Contoh menggunaan 


   set()

mari kita coba, buat file baru dan ikuti kode berikut 


  teman = ["akmal","adi","adi"]

  print set(teman)

Hasilnya

data adi yang tadinya ada dua akan menjadi satu, karena telah dihapus oleh fungsi set() sekian pembelajaran kali ini, dan silahkan pelajari tutorial selanjutnya Lanjut
