memahami try exception

apa itu try..except di python? try..except adalah fungsi untuk penanganan error pada code (exception handling) logikanya:

contoh dalam program:


  try:
    # jalankan code ini...
  except:
    # jalankan ini ketika ada pengecualian/error

Paham gak? kalau belum paham mari kita coba, buat file baru bernama latihan.py dan ikuti kode ini


  try:
      print hello
  except:
      print "kode error!"

ini sebenarnya agak bodoh, hehe... kenapa?karena kita mencetak variabel hello sedangkan kita tidak mempunyai variabel hello,tentu ini akan error.tapi gak papa ini hanya untuk pembelajarankalau jalankan maka akan masuk kebagian except dan akan menampilkan pesan "kode error!"

tapi, bagaimana jika kodenya banyak? sedangkan kita tidak tahu di bagian mana ada error...tinggal gunakan fungsi ExceptionContoh:


  try:
      print hello
  except Exception as errornya:
      print "Error bagian: " + str(errornya)
