percabangan if..else

apa itu percabangan ? percabangan itu seperti logika "jika..maka", Percabangan akan membuat program menentukan tindakan sesuai dengan logika/kondisi yang telah kita berikan.pastikan sebelumnya sudah memahami tentang "6 jenis operator pada python". percabangan di python bisa kita tulis seperti ini 


  i = 10

  if i == 10:
    # jika variabel i sama dengan 10 maka akan masuk kesini
  else:
    # jika tidak sama, maka akan masuk kesini

biar cepat paham, mari kita latihan. silahkan buat file baru bernama latihan_percabangan.py dan ikuti kode berikut


  umur = input("berapa umur kamu?: ")

  if umur == 18:
    # kondisi pertama
    print "umur kita sama"
  else:
    # kondisi kedua
    print "umur kita beda"

Hasilnya:

apa artinya?, artinya jika inputan umur sama dengan 18 maka akan masuk ke kondisi pertama,jika tidak, maka akan masuk ke kondisi kedua

percabangan if..elif..else

percabangan if..elif..else ini digunakan apabila membutuhkan lebih dari 2 keputusan,ikuti saja kode ini dulu, nanti saya jelaskan


  umur = input("berapa umur kamu?: ")

  if umur > 18:
    # jika umur lebih besar dari 18, maka akan masuk kesini
    print "umur kamu di atas 18"

  elif umur < 18:
    # jika umur lebih kecil dari 18, maka akan masuk kesini
    print "umur kamu di bawah 18"

  else:
    # jika kedua kondisi di atas tidak terpenuhi, maka akan masuk kesini
    print "umur kamu 18"

Hasilnya:

jika kondisi pertama tidak terpenuhi, maka ia akan mengecek kondisi kedua, jika kondisi kedua tidak cocok juga, maka akan masuk kebagian elsesekian pembelajaran tentang percabangan kali ini, semoga sudah paham.kalau belum paham juga, bisa contact saya, kalau sudah paham silahkan pelajari tutorial selanjutnya Lanjut
